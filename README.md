# Neovim Config

My personal neovim config.

---

## Installation

### 1. Clone this repo

```bash
git clone git@github.com:kotayam/neovim-config.git
```

### 2. Run setup.sh

```bash
cd neovim-config
chmod +x setup.sh
./setup.sh
```
